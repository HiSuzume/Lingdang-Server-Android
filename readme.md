## 🔔铃铛 Server
### 概述
一个用于简易网站搭建的轻量级Android程序😁

后续会更新更实用的功能，欢迎提交合理的Issues~😉

启动流程:主入口→前台服务→HTTP服务线程→用户自主关闭或后台被关闭
### 背景
曾经我有建站需求，但是现有的软件有些功能全但太臃肿，有些很轻量但无法保活，有些内存占用高......

因此我自己开发了铃铛Server，供有需求的人和我使用😂
### 特点
* 🚀轻量
* 💡防掉线(因设备而异，建议添加白名单)
* 🗝Lua脚本支持
### 鸣谢
