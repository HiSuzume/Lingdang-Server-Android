## 铃铛
#### 概述
铃铛 是一个使用 NanoHTTPD ，运行在Android系统的轻量Http服务的实现哦!

#### 特点
* 轻量，占用小
* 防掉线(最保险:关闭省电模式并添加后台白名单)
* 可以执行 Lua 脚本
* 断点传输(音视频已测试)
* 无缓存，真正小而美!
