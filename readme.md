### 铃铛
#### 概述
铃铛 是一个运行在Android系统的轻量Http服务器软件哦!
#### 背景
找不到合适的轻量级Http服务软件，于是就自己开发了~~明明是给自己水时间找理由(bushi)~~
#### 特点
* 🚀轻量，占用小
* 💡防掉线(最保险:关闭省电模式并添加后台白名单)
* ⌨Lua脚本
* 📝断点传输(音视频已测试)
* 🚫无缓存，真正小而美!
#### 使用的源代码
* [NanoHTTPD](https://github.com/NanoHttpd/nanohttpd)
* LuaJ++
